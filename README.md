# Deep-learning-Wild-Fire-Prediction-

FireCube Dataset Preparation.ipynb: Preprocesses and structures the FireCube dataset for input into the deep learning model, including data cleaning, formatting, and feature preparation.

Fire Prediction Deep Learning Model.ipynb: Implements and runs the deep learning model for wildfire prediction using the prepared dataset, including training, validation, and evaluation.
